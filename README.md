# InDesign Multi Search & Replace

**DE** | [EN](#english)

---

## Deutsch

Ein Adobe InDesign Script für mehrfaches Suchen und Ersetzen in einem Durchgang.

### Features

- 5 Suchen-und-Ersetzen-Felder gleichzeitig
- **Nicht-modales Fenster** – Dokument bleibt bedienbar, Text kann kopiert werden
- Optionen für Groß-/Kleinschreibung und Ganzes Wort
- Detaillierter Ergebnisbericht
- GREP-basiert für zuverlässige Ergebnisse

### Installation

1. Script herunterladen (`MultiSearchReplace.jsx`)
2. In den InDesign Scripts-Ordner kopieren:
   - **macOS:** `~/Library/Preferences/Adobe InDesign/[Version]/[Sprache]/Scripts/Scripts Panel/`
   - **Windows:** `C:\Users\[User]\AppData\Roaming\Adobe\InDesign\[Version]\[Sprache]\Scripts\Scripts Panel\`
3. In InDesign: Fenster → Hilfsprogramme → Skripte
4. Doppelklick auf das Script

### Verwendung

1. Dokument öffnen
2. Script starten
3. Suchbegriffe und Ersetzungen eingeben
4. Optional: Optionen aktivieren
5. "Ersetzen" klicken

### Screenshot

Das Script öffnet ein schwebendes Fenster mit 5 Zeilen für Suchen/Ersetzen-Paare.

## English

An Adobe InDesign script for multiple search and replace operations in one go.

### Features

- 5 search-and-replace fields at once
- **Non-modal window** – Document stays accessible, text can be copied
- Options for case sensitivity and whole word matching
- Detailed result report
- GREP-based for reliable results

### Installation

1. Download the script (`MultiSearchReplace.jsx`)
2. Copy to InDesign Scripts folder:
   - **macOS:** `~/Library/Preferences/Adobe InDesign/[Version]/[Language]/Scripts/Scripts Panel/`
   - **Windows:** `C:\Users\[User]\AppData\Roaming\Adobe\InDesign\[Version]\[Language]\Scripts\Scripts Panel\`
3. In InDesign: Window → Utilities → Scripts
4. Double-click the script

### Usage

1. Open a document
2. Run the script
3. Enter search terms and replacements
4. Optional: Enable options
5. Click "Ersetzen" (Replace)
